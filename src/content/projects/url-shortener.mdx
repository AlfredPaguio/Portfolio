# URL Shortener

This project is a simple and straightforward URL shortener service that allows you to create short links quickly and easily. With this project, you can shorten long URLs into concise, shareable links that are perfect for social media, email, and other online platforms.

## Features

* **Simple Interface**: provides an intuitive user interface for creating short links in just a few clicks.
* **Randomized Short URLs**: generates unique, randomized 13-character short URLs using nanoid for each long URL you provide, ensuring their uniqueness and security.
* **Click Count Tracking**: Keep track of the number of clicks on your short links.
